HOW TO USE THE APPLICATION
================================================================================

STEP 1: Upload Image
  - Click the upload box or drag an image
  - Supports JPG, PNG formats
  - Any size (optimized automatically)

STEP 2: Watch Processing
  - See animated neural network scanner
  - Processing takes 2.5 seconds
  - "Analyzing..." message displayed

STEP 3: View Results
  - Left panel: Image with bounding boxes
  - Right panel: Detection report list
  - Confidence scores shown
  - Status indicators displayed

STEP 4: Analyze Another Image
  - Click "Reset & Analyze New Image"
  - Upload starts fresh
  - Process repeats
